## Xml2Html

使用Python将XML文件转换为HTML文件
主要使用目的：工作项目中需要编写googleTest测试用例，googleTest测试用例需要使用XML格式，但是XML格式不方便查看，因此编写此工具将XML文件转换为HTML文件，方便查看。


### Installation
```bash
pip3 install xml2html
```
### Usage

```bash
xml2html -f <xml file path> -o <output html file>
```
